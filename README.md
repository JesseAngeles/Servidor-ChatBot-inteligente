# Servidor ChatBot Inteligente

Servidor backend en TypeScript para un chatbot con procesamiento de dataset en CSV y persistencia en MongoDB.

## Tecnologías

- TypeScript
- Node.js
- Express
- MongoDB (Mongoose)
- CSV Parser

## Instalación

```bash
npm install
```

Crear un archivo `.env` con la URI de MongoDB:

```
MONGO_URI=tu_uri_de_mongodb
```

## Ejecución

```bash
npm start
```

## Estructura

```
src/          # Código fuente
dataset/      # Dataset en formato CSV
Documentation/
```
